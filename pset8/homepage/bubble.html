Bubble Sorting

Bubble sorting works by looping through a list or an array and swapping elements that are out of order. If the ith element is greater than the i + 1 element the algorithm requires a swap. This process is repeated until the list is ordered.

Bubble Sorting Steps

Step 1 :

Start by looping over every element of the list or array. Bubble sort requires two loops, thus the first step establishes the outer loop. The outer loop will keep track of say an integer i.

Step 2 :

Once the outer loop has been created, the second loop that will allow the comparison of each element is created. The inner loop will keep track of say an integer j.

Step 3 :

For each iteration of the inner loop, compare the jth element with the j + 1 element. If the jth element is greater, swap both values in the list

Step 4 :

Repeat step 3 until the list is fully ordered. Thus, in the worst case scenario, where the list is in reverse order, this process has an exponential time complexity.
