Welcome to Sorting Algorightms!

In computer science, there are many sorting algorithms that allow us to order arrays and lists according to a specified condition! Sorting algorithms offer different time complexities, thus it is important to choose one that is suitable for your needs. Often in computer science, there is a trade-off between time and space, where algorithms that are more time effecient require more space in memory and vice-versa!

Types of Sorting Algorithms

Bubble Sort

  • Simple sorting algorithm
  • Compare adjacent elements
  • Swap elements if out of order
  • Repeat until list is ordered

Merge Sort

  • Seperates list in two halves
  • Orders left then right half
  • Repeats until half is of size 1
  • Merges ordered halves

Insertion Sort

  • Loop over list starting at index 1
  • Compare element by looping again
  • Insert element at ordered position
  • Repeat until list is ordered
